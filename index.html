What is a component?

com · po · nent

/kəmˈpəʊnənt/

reusable Vue instances with a name

Example


            <!-- counter-button -->
            <button @click="count++">
              You clicked me {{ count }} times.
            </button>
          

            {
              data: function() {
                return {
                  count: 0
                };
              }
            }
