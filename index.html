What is a component?

com · po · nent

/kəmˈpəʊnənt/

reusable Vue instances with a name

Basics

Example


            <!-- counter-button -->
            <button @click="count++">
              You clicked me {{ count }} times.
            </button>
          

            {
              data: function() {
                return {
                  count: 0
                };
              }
            }

Usage


            <counter-button></counter-button>
            <counter-button></counter-button>

Registration


            Vue.component('button-counter', {
              data: function() {
                return {
                  count: 0
                };
              },
              template: `
                
              `;
            });
          

            <counter-button></counter-button>

Single File Components

Structure


            <template>
              <button @click="count++">
                You clicked me {{ count }} times.
              </button>
            </template>

            <style>
              button {
                /* ... */
              }
            </style>

            <script>
              export default {
                name: 'counter-button',
                data() {
                  return {
                    count: 0
                  };
                }
              }
            </script>

Component Tree

Props

Example


            <template>
              <h3>{{ title }}</h3>
            </template>

            <script>
              export default {
                name: 'blog-post',
                props: ['title']
              }
            </script>

Usage


            <blog-post title="Awesome blog"></blog-post>
            <blog-post title="Yet another awesome blog"></blog-post>
